# CronTab Components

Модульная структура страницы CronTab с lazy loading и современным дизайном.

## Структура

```
CronTab/
├── TaskEditor.tsx       - Форма создания/редактирования задач
├── TaskList.tsx         - Список задач с управлением
├── ExecutionHistory.tsx - История выполнения
└── index.ts            - Экспорты компонентов и типов
```

## Особенности

- **Lazy Loading**: Все компоненты загружаются по требованию
- **Skeleton Loaders**: Плавная загрузка с анимацией
- **Без бордеров**: Чистый дизайн в стиле DataStudio/Dashboard
- **Темная тема**: Backdrop blur и полупрозрачные фоны
- **TypeScript**: Полная типизация всех компонентов

## Использование

```tsx
import { CronTab } from "@/pages/CronTab";

// Компонент автоматически загружает подкомпоненты
<CronTab />
```

## Типы

Все типы экспортируются из соответствующих компонентов:
- `FormState` - состояние формы редактора
- `CronTask` - структура задачи
- `CronExecutionRecord` - запись выполнения
